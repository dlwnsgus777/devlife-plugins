tdd-team
v0.5.0
에이전틱 Red-Green-Refactor TDD 사이클로 기능을 개발한다. 테스트 작성 → 실패 확인 → 최소 구현 → 리팩터링을 자동 반복한다.

branch-review
현재 브랜치의 모든 변경사항을 main 대비 분석하고 4개 차원(컨벤션·테스트·도메인·설계)으로 점수화한 리포트를 저장한다.

plan-creator
Spring Boot API 기능 구현을 위한 구조화된 계획 문서 작성. Tidy First 정비 판별 → 도메인 컨텍스트 수집 → 8섹션 계획서 생성.
계획 작성해줘구현 계획계획 MD로 정리
📑
prd-creator
대규모 피처·에픽 단위의 PRD 작성. 전체 범위·목표 정의 → plan-creator 연계 서브태스크 분해.
PRD 작성해줘에픽 계획전체 계획 세워줘
📄
pdf-to-prd
PDF를 macOS Vision/PDFKit으로 텍스트 추출한 뒤 prd-creator 워크플로우로 PRD 문서를 생성한다.
PDF 분석해줘PDF 보고 PRD 만들어줘
