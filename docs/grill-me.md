# grill-me

계획이나 설계에 대해 집중적으로 질문하여 약점을 찾아냅니다.  
설계 결정 트리의 모든 분기를 하나씩 검증하고, 각 질문마다 추천 답안을 함께 제시합니다.

## 언제 사용하나요?

- 계획이나 설계를 스트레스 테스트하고 싶을 때
- "내가 놓친 부분이 있나?" 확인이 필요할 때
- 설계 결정의 근거를 명확히 하고 싶을 때
- PR 전에 설계를 한 번 더 검증하고 싶을 때

## 트리거 문구

```
"grill me"
"나 심문해줘"
"계획 검증해줘"
"설계 파고들어줘"
"약점 찾아줘"
```

## 동작 방식

- 질문은 **한 번에 하나씩** 순차적으로 제시
- 코드베이스로 답할 수 있는 질문은 직접 탐색하여 답변
- 각 질문마다 **추천 답안**을 함께 제공
- 설계 결정 트리의 모든 분기를 해결할 때까지 계속
